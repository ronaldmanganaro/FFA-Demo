Known Issues(email me at ronaldmanganaro@gmail.com with issues not listed)
Memory issue on Firefox Workaround: Initially Firefox worked but if it does not use Edge/Chrome.
Can walk off screen Workaround: make sure that you use the full screen button under the demo
Can't continue past character select This is not an error just make sure that there are 2 players not including someone using the mouse/gamepad.
Also make sure to enter names by clicking sel and then save after typing in a name.
Blurred arrow buttons This does not stop the buttons from working to select a character so none issue.
